> **문서 버전:** v0.1 (초안)
> **작성일:** 2026-05-27

## 1. 개요

### 1.1 배경 및 목적

말씀을 듣기만 하는 것이 아니라 실제로 행하는 삶을 살아내기 위해 (약 1:22), 교회 공동체 구성원 개개인이 사명·비전·핵심가치에 기반한 연간 목표를 설정하고, 일별 실천 여부를 기록·집계하여 성장을 추적할 수 있는 시스템이 필요하다.

### 1.2 대상 사용자

| 역할            | 설명                                                                            |
| --------------- | ------------------------------------------------------------------------------- |
| **일반 사용자** | 공동체 구성원. 자신의 일별 목표 달성 여부를 체크하고 월별/연별 현황을 확인한다. |
| **관리자**      | 사명, 비전, 핵심가치, 목표 항목 전체를 설정·수정한다.                           |

> 모든 구성원은 **동일한 사명·비전·핵심가치·세부 목표 항목**을 공유한다. 개인별로 달리 설정되는 것은 월별/연별 **목표 수치(달성 횟수)** 뿐이다.

### 1.3 핵심 가치 (시스템 설계 원칙)

본 시스템은 아래 핵심 가치를 시스템 설계에도 반영한다.

| 핵심 가치 | 성경 구절 | 시스템 반영                                       |
| --------- | --------- | ------------------------------------------------- |
| 정직      | 엡 4:25   | 조작 불가, 과거 기록 수정 이력 보존               |
| 충성      | 고전 4:2  | 빠짐없는 일별 체크 독려, 연속 달성 표시           |

| **기록**  | 합 2:2    | 달려가면서도 읽을 수 있게 — 모바일 친화적 체크 UI |

## 2. 사명·비전·핵심가치 (콘텐츠 정의)

### 2.1 사명 (Mission)

말씀을 듣기만 하여 자신을 속이는 자가 되지 않고 말씀을 행하는 삶을 살아낸다
말씀을 행하는 자가 되고, 듣기만 하여 자신을 속이는 자가 되지 말라 (약 1:22)

영어로 된 강의안을 잘 만들고 외워서 강의하는 '어성경' 전문강사가 된다.

## 4. 주요 기능 요구사항

### 4.1 랜딩 페이지 (공통)

- **사명·비전·핵심가치**를 첫 화면에 명확하게 표시한다.
- 목표(4대 영역)는 랜딩 페이지에서 분리하여 별도 메뉴 또는 탭으로 제공한다.
- 로그인 후 본인의 오늘 날짜 기준 달성 현황 요약(미체크 항목 수)을 상단에 표시한다.

### 4.2 관리자 화면

#### 4.2.1 항목 관리

- 사명, 비전, 핵심가치, 4대 영역의 대항목·세부항목을 **추가·수정·삭제**할 수 있다.
- 세부 항목에는 다음 속성을 설정할 수 있다.
  - 이름, 설명, 주기(매일/매주/매달/연간), 기본 목표 수치, 활성화 여부
- 항목 변경 이력을 보존한다 (삭제 시 소프트 삭제).

#### 4.2.2 사용자 관리

- 사용자 등록·비활성화를 할 수 있다.
- 사용자별 월별 목표 수치를 개별 설정할 수 있다 (기본값은 항목별 기본 목표 수치).

### 4.3 일별 달성 체크

- 날짜 선택(기본: 오늘) 후 **전체 세부 항목 목록**을 표시한다. 주기(매일/매주/매달/연간)에 관계없이 모든 항목을 매일 체크할 수 있다.
- 각 항목을 **체크/언체크** 할 수 있다.
- **동일 항목은 하루에 1회만 카운트**된다. 같은 날 같은 항목을 중복 체크하는 것은 불가하며, 체크 = 1회, 미체크 = 0회로 집계된다.
- 과거 날짜 체크는 허용하되, 수정 시 이력을 기록한다.
- 모바일에서도 빠르게 체크할 수 있는 UI를 제공한다 (합 2:2 — 달려가면서도 읽을 수 있게).

**주기별 집계 기준:**

| 주기 | 집계 방식                    |
| ---- | ---------------------------- |
| 매일 | 해당 날짜 체크 여부 (1 or 0) |
| 매주 | 주 내 체크된 날 수 누적      |
| 매달 | 월 내 체크된 날 수 누적      |
| 연간 | 연 내 체크된 날 수 누적      |

### 4.4 월별/연도별 목표 입력 화면

- 관리자 또는 본인이 월별 목표 수치를 입력할 수 있다.
- **기본값:** 해당 월의 일수 − 4 (여유분 반영).
- **12달 연간 합계**는 각 달의 월별 목표 수치의 합으로 자동 계산된다.
- 개인별로 목표 수치를 다르게 설정할 수 있다.
- 목표 수치 변경 이력을 보존한다.

### 4.5 달성 현황 화면

#### 월별 현황

- 월 단위로 조회: 세부 항목별 **당월 달성 횟수**와 **당월 목표 수치** 표시.

#### 누적 진척률 (핵심 지표)

- **달성률의 기준은 "1년 전체 목표 수치 합계 대비 연초부터 현재까지의 누적 달성 횟수"**이다.
- 누적 진척률(%) = (연초~현재까지 누적 달성 횟수 합계 / 연간 목표 수치 합계) × 100
- 예: 연간 목표가 365회인 항목에서 1월~5월까지 150회 달성 → 누적 진척률 41.1%
- 항목별 누적 진척률과 **전체 항목 통합 누적 진척률**을 함께 표시한다.
- 현재 날짜 기준 **기대 진척률** (오늘이 연중 몇 %인지)을 함께 표시하여 앞서가는지/뒤처지는지 한눈에 파악 가능하게 한다.

#### 연도별 현황

- 12개월 월별 달성 횟수와 누적 진척률 추이를 일괄 표시 (트렌드 확인).

## 5. 비기능 요구사항

| 항목        | 요구사항                                                                  |
